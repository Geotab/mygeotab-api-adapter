## About v5.0.0

Version **5.0.0** is the **initial release** of the Geotab DIG Adapter. While the MyGeotab API Adapter pulls data *out of* MyGeotab, the Geotab DIG Adapter pushes telemetry data from custom (non-Geotab) telematics devices *into* MyGeotab via the [Data Intake Gateway (DIG)](https://github.com/Geotab/data-intake-gateway). The two adapters can be used independently or together — they share the same `geotabadapterdb` database but operate in separate schemas.

There is no prior version to upgrade from. For installation instructions, see [Section 1 Quick Start Guide](../../README.md#1-quick-start-guide) in the README.

For an overview of what's included in v5.0.0, see the [Change Log](../../CHANGELOG.md).
